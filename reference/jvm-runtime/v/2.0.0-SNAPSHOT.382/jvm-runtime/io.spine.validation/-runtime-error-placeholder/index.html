RuntimeErrorPlaceholder

enum RuntimeErrorPlaceholder : Enum<RuntimeErrorPlaceholder> (source)

A template placeholder that can be used in error messages.

Enumerates placeholder names that can be used within Protobuf definitions. Each validation option declares the supported placeholders within options.proto.

Important Note: this enum is an exact copy of io.spine.validation.java.ErrorPlaceholder. Please keep them in sync. Take a look at docs to the original enum for details.
