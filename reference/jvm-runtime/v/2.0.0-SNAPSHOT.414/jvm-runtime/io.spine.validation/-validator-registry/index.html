ValidatorRegistry

@ThreadSafe
object ValidatorRegistry(source)

A registry for custom validators of Protobuf messages.

This registry allows for dynamic registration and retrieval of custom validators implementing the MessageValidator interface.

It supports several validators per message type and provides an API for validating messages by applying all associated validators.

The registry also automatically loads validators from the classpath using the ServiceLoader mechanism.
