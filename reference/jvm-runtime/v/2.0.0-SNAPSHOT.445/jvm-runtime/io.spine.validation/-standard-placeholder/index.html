StandardPlaceholder

enum StandardPlaceholder : Enum<StandardPlaceholder> (source)

The standard set of Placeholders that can be used in error messages.

Enumerates placeholder names that can be used within Protobuf definitions. Each validation option declares the supported placeholders. Take a look at options.proto for examples.

The enum is used by the compiler model and Java renderer when validating and rendering built-in option error messages.
