ValidationWarnings

class ValidationWarnings : GeneratedMessage, ValidationWarningsOrBuilder
Per-kind toggles for warnings emitted by the Java target of
the Validation Compiler.

The Gradle plugin writes this message to the Compiler settings directory.
Each flag is independent and defaults to `true` (warning enabled) on the
DSL side; the Gradle plugin always writes an explicit value, so the
renderer never has to distinguish "field absent" from "field set to false".

If no settings file is present at all (e.g., the renderer is invoked
outside of the Validation Gradle plugin), the renderer treats every flag
as enabled — preserving the historical "always warn" behavior.
