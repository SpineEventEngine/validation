    Per-kind toggles for suppressing warnings emitted by the Java target
of the Validation Compiler.

Each flag is independent. A flag set to `true` suppresses the
corresponding warning; the proto3 default `false` keeps the warning
emitted, which matches the renderer's "warn unless told otherwise"
behavior when no settings are present at all.
